# Usage Guide for ADMIN Management System

## Overview
The ADMIN Management System is a comprehensive web application for organizational management, including personnel, vehicle tracking, invoicing, banking, taxes, and more. This guide focuses on how to use the application once it's deployed and accessible online.

## Accessing the Application
1. Open your web browser and navigate to the deployed application URL (https://gregarious-heliotrope-9c72b4.netlify.app).
2. If not logged in, you'll be redirected to the sign-in page.

## User Authentication
- **Sign Up**: Create a new account using the signup form (if registration is enabled).
- **Sign In**: Log in with your username/email and password.
- The system supports role-based access control, so your dashboard and available features depend on your assigned role.

## Main Modules and Features

### Dashboard
- Provides an overview of key metrics, alerts, and quick access to main functions.
- Role-based dashboard displaying relevant information based on user permissions.

### Personnel Management
- **Users**: Manage employees - add, edit, view personnel details, roles, and permissions.
- **Absences**: Track employee absences, including dates, reasons, and approvals.
- **Contracts**: Manage employment contracts, including start/end dates, terms, and renewals.
- **Bonuses**: Record and track employee bonuses and incentives.
- **Medical Records**: Maintain confidential medical information for employees.
- **Sanctions**: Document disciplinary actions and sanctions.
- **Affectations**: Manage employee assignments and job placements.

### Vehicle Management
- **Vehicles**: Add and manage fleet vehicles, including specifications, registration, and status.
- **Fuel Management**: Track fuel consumption, costs, and efficiency.
- **Maintenance**: Schedule and record vehicle maintenance activities.
- **Repairs**: Document repair work, costs, and parts used.
- **Expenses**: Track all vehicle-related expenses.
- **Authorizations**: Manage vehicle usage permissions and authorizations.
- **Garages**: Maintain information about service providers and garages.
- **Interventions**: Log emergency or special vehicle interventions.
- **Payment Cards**: Manage fuel cards and payment methods.
- **Pieces**: Inventory spare parts and components.
- **Reforms**: Track vehicle modifications and upgrades.
- **State Vehicles**: Monitor vehicle condition and operational status.

### Financial Modules
- **Banks**: Manage bank accounts, balances, and basic banking information.
- **Bank Transactions**: Record and track all bank transactions, deposits, and withdrawals.
- **Invoices**:
  - Company Expenses: Track business expenses and reimbursements.
  - Professional Services: Manage invoices for professional services.
  - Other Invoices: Handle miscellaneous invoicing.
- **Taxes**: Manage tax declarations, calculations, and filings.

### Equipment and Software Management
- **Equipment**: Track laboratory or operational equipment.
  - Assignments: Assign equipment to users or departments.
  - Calibrations: Schedule and record equipment calibrations.
  - Maintenance: Plan and log maintenance activities.
  - Expenses: Track equipment-related costs.
  - Repairs: Document repair work and costs.
  - Revisions: Manage equipment audits and revisions.
- **Software**: Manage software licenses and installations.
  - Assignments: Assign software to users.
  - Contracts: Track software licensing agreements.
  - Maintenance: Schedule software updates and patches.
  - Contacts: Maintain vendor contact information.

### Other Modules
- **Contacts**: Manage contact information for clients, vendors, and partners.
- **Documents**: Upload, organize, and manage important documents.
- **Alerts**: View system notifications and overdue items.
- **Overdue Items**: Track items requiring attention (contracts expiring, maintenance due, etc.).
- **Business**: Manage business entities and relationships.
- **Organigramme**: View organizational structure and reporting relationships.
- **Registers**: Maintain transaction registers and logs.
- **Offers**: Manage business proposals and quotes (AMI, DAO, Devis).

## How to Use the Application

### Navigation
- Use the main menu/sidebar to navigate between modules.
- Each module has its own section with list views, detail views, and forms.

### Adding New Records
1. Navigate to the desired module.
2. Click "Add", "Create", or "+" button.
3. Fill in the required fields in the form.
4. Upload any necessary files (documents, images).
5. Click "Save" or "Submit".

### Editing Existing Records
1. Find the record in the list view.
2. Click on the record or use the "Edit" button.
3. Modify the information as needed.
4. Save changes.

### Searching and Filtering
- Use search bars to find specific records.
- Apply filters to narrow down results (date ranges, status, categories).

### Reports and Exports
- Many modules offer PDF report generation.
- Export data to Excel for further analysis.
- Use the "Export" or "Download" buttons.

### File Uploads
- The system supports file uploads for documents, images, and other attachments.
- Supported formats: PDF, DOC, XLS, images (JPG, PNG), etc.
- Files are securely stored and accessible through the application.

## Data Retrieval and Access

### Accessing Your Data
Since the application is online with an active database:

1. **Through the Web Interface**: All data you enter is immediately stored and can be accessed through the application's modules. Use search and filter functions to retrieve specific information.

2. **Reports and Exports**: Generate PDF reports or export data to Excel/CSV for offline access.

3. **API Access**: For programmatic access (if enabled), use the provided API endpoints to retrieve data programmatically.

### Data Security and Privacy
- All data is encrypted and securely stored.
- Access is controlled by user roles and permissions.
- Personal and sensitive information is protected according to data protection standards.

## Troubleshooting
- If you encounter issues, try refreshing the page.
- Clear browser cache if experiencing display problems.
- Contact your system administrator for technical support.
- Check your internet connection for online features.
